    **1. Create `apps/web/app/components/roadmap/RoadmapNodeTree.tsx`** — node tree visualization per D-08, UX-01, UI-SPEC Screen 3:

    Props:
    ```typescript
    interface NodeTreeProps {
      nodes: RoadmapNode[];
      completedLessonIds: string[];
      roadmapId: string;
      complexity: "linear" | "branching";
    }
    ```

    **Node state computation** (per Pitfall 7 — compute client-side from completedLessonIds):
    - A node is "completed" if its lessonId is in completedLessonIds
    - A node is "locked" if any of its prerequisites are NOT completed
    - A node is "available" if all prerequisites are completed and it is NOT completed
    - First node (order 0, no prerequisites) is always available

    **Linear layout** (complexity === "linear"):
    - Single vertical column of nodes
    - Nodes connected by a 2px vertical line in `border` color
    - Nodes stacked with `gap-8` (32px, xl token)

    **Branching layout** (complexity === "branching"):
    - Tree structure — indent child nodes 32px (`pl-8`) per prerequisite level
    - Parent-child connectors: 2px solid lines (CSS borders or SVG)
    - Root nodes (no prerequisites) at left margin
    - Children indented right

    **Node rendering** per UI-SPEC:
    - Shape: `Card` component, full-width minus 32px margin (`mx-4`), `min-h-12` (48px tap target)
    - **Locked:** `bg-card`, `border-border`, `Lock` icon (16px, muted-foreground), title in `text-muted-foreground`, `aria-disabled="true"`, `tabIndex={-1}`
    - **Available:** `bg-card`, `border-border`, title in `text-foreground`, no lock icon
    - **In progress:** `bg-card`, `border-primary` (accent), title in `text-foreground`
    - **Completed:** `bg-card`, `border-border`, title in `text-foreground`, `CheckCircle` icon (accent, 16px) right-aligned

    Each node is a `<button>` with `aria-label="{title} — {state}"` (per UI-SPEC accessibility).

    - Available/in-progress/completed nodes: navigate to `/roadmaps/${roadmapId}/lessons/${lessonId}` on tap
    - Locked nodes: show inline tooltip "Complete previous lessons to unlock" (1.5s, auto-dismiss) using shadcn Tooltip — NO navigation
