<task type="checkpoint:human-verify" gate="blocking">
  <name>Task 1: Verify complete Phase 2 end-to-end flow</name>
  <what-built>
    Complete AI content pipeline: buddy chat with streaming + roadmap generation, roadmap list and node tree visualization, lesson view with Markdown content and quiz, practice quiz mode, and RAG Q&A (in-lesson bottom sheet + roadmap-level tab). All screens are mobile-first with 5-tab navigation.
  </what-built>
  <how-to-verify>
    Start the dev server: `cd apps/web && npm run dev`

    **1. Navigation (D-06):**
    - Verify bottom nav shows 5 tabs: Home, Chat, Roadmaps, Battle, Profile
    - Verify desktop sidebar shows same 5 items
    - Verify active tab indicator works correctly

    **2. Buddy Chat (D-01, D-02, CONT-05):**
    - Navigate to Chat tab
    - Verify empty state: "What do you want to learn?" heading
    - Type a conversational message (e.g. "Hello") — verify streaming response appears with AI bubble styling
    - Verify response starts streaming within ~2 seconds

    **3. Roadmap Generation (CONT-01, CONT-06, D-03, D-04):**
    - Type "Create a roadmap for learning Python" in chat
    - Verify progress bubble appears with 3 steps: "Analyzing topic...", "Building roadmap...", "Generating lessons..."
    - Wait for generation to complete (10-30 seconds)
    - Verify "Your roadmap is ready." message with "View roadmap" button appears
    - Verify you are NOT auto-navigated (D-16)

    **4. Roadmaps List (D-07):**
    - Navigate to Roadmaps tab
    - Verify the generated roadmap appears in the list with title and progress bar
    - Verify progress shows "0 of N lessons complete"

    **5. Node Tree (CONT-02, UX-01, D-08):**
    - Tap the roadmap to open detail view
    - Verify node tree renders with lesson nodes
    - Verify first node is "available" (tappable), subsequent nodes are "locked"
    - Tap a locked node — verify tooltip "Complete previous lessons to unlock" appears

    **6. Lesson View (CONT-03, CONT-04, D-10):**
    - Tap the first available lesson node
    - Verify lesson content renders as formatted Markdown (headings, paragraphs, lists)
    - Verify quiz section appears below content with "Knowledge Check" divider

    **7. Quiz Feedback (UX-02, D-11, D-12):**
    - Answer a quiz question by tapping an option
    - Verify instant feedback: correct = blue border, wrong = red border + correct answer highlighted
    - Verify explanation text appears below
    - Complete all questions, verify "Complete lesson" button appears

    **8. Lesson Completion:**
    - Tap "Complete lesson"
    - Verify navigation back to roadmap detail
    - Verify the completed lesson node now shows CheckCircle icon
    - Verify next lesson is now "available" (unlocked)

    **9. In-Lesson Q&A (QNA-01, D-05):**
    - Open a lesson, tap "Ask AI" button
    - Verify bottom sheet opens at ~60% height
    - Ask a question about the lesson content
    - Verify answer appears with citation link(s) [Lesson N: Title]
    - Tap a citation — verify it closes sheet and navigates to cited lesson

    **10. Roadmap Q&A (QNA-02, D-14):**
    - Go to roadmap detail, tap "Q&A" tab
    - Ask a question about the roadmap content
    - Verify answer with citations
    - Verify citations link to correct lessons

    **11. Practice Quiz (D-13):**
    - Navigate to practice quiz for a roadmap with completed lessons
    - Verify questions appear (randomized from completed lessons)
    - Answer all questions, verify score summary

    **12. Mobile Layout:**
    - Resize browser to mobile viewport (375px width)
    - Verify all screens are usable on mobile
    - Verify 48px minimum tap targets on nav tabs, quiz options, buttons
  </how-to-verify>
