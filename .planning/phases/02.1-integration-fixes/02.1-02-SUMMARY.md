## Changes Made

### Task 1: Q&A Citation Field Rename and Shape Alignment

- Renamed response field from `sources` to `citations` to match frontend `QAResponse` interface
- Changed citation object shape from `{ lessonId, title, displayText }` to `{ lessonId, lessonTitle, lessonOrder }`
- Updated D1 query to select `schema.lessons.order` alongside `id` and `title`
- Replaced `lessonTitleMap` (Map<string, string>) with `lessonMap` (Map<string, { title, order }>)
- Updated context chunk builder to use new map accessor pattern

### Task 2: AuthVariables Generic and API Client Interface Alignment

- Created and exported `AuthVariables` type from `auth-guard.ts` with `userId: string` and `session: Record<string, unknown>`
- Added `Variables: AuthVariables` to roadmapRoutes Hono generic parameter
- Removed all 6 `as string` casts on `c.get("userId")` throughout roadmaps.ts
- Updated `RoadmapListItem` interface: removed `description`/`updatedAt`, added `topic`/`complexity`/`status`
- Updated `RoadmapDetail` interface: removed `description`/`totalLessons`/`completedLessons`, added `userId`/`topic`/`complexity`/`status`/`workflowRunId`/`completedLessonIds`
- Kept `totalLessons`/`completedLessons` in `RoadmapListItem` (list endpoint returns them)
